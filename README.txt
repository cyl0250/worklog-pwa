# 작업일지 PWA (안드로이드용)

이 ZIP에는 프론트(PWA) 파일만 들어있습니다:
- index.html
- app.js
- manifest.json
- README.txt

## 해야 할 일
1) app.js 맨 위 GAS_URL을 본인 Apps Script Web App URL로 변경
2) GitHub Pages(또는 다른 HTTPS 호스팅)에 업로드
3) 휴대폰 크롬에서 열고 '홈 화면에 추가'

## app.js 설정
- TOKEN은 기본값 worklog2026로 들어있습니다.
- Apps Script Code.gs의 API_TOKEN이 worklog2026인지 확인하세요.
